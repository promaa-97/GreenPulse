Website Template and Navigation

Technical Description

Created using HTML and CSS. I used div tags for content organization. Anchor tags used to link different pages to Navigation Bar. I gave a fixed possition for navigation bar using css. img tags used to import images to navigation and footer section. Used 5 different sections for footer menu. And also included external links with target attribute to view in a new window.

Accessibility

Applied :hover effects for interacrtive links in Navigation and Footer. Template provided with a default background color and a font. Gave 20% margin to both left and right for container divition to improve readabillity. I also added alt attribute for all images to provide descriptive text for screen readers.

Challenges and Lessons Learned

Initially, the content sections were misaligned when viewed on different screen sizes. To resolve this, I used @media for 1024px and 768px for better responsiveness and adjusted margins and paddings accordingly. I learned the importance of responsive design and how Flexbox/Grid can simplify layouts. I learned to use proper form validation techniques improves both user experience and security.

Compliance

To ensure compliance with Janet regulations and WCAG standards, I followed best practices for web development:

  • Used proper syntax and validated code using the W3C HTML and CSS validator tools.
  • Ensured no user data is stored insecurely and followed privacy best practices.
  • Maintained a logical page structure with clear, accessible navigation links.
  • Used responsive design principles to make the website accessible across devices.
